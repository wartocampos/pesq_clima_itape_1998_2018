# Projeto Pesquisa Climatica Itapetininga 1998 - 2018
Este projeto tem como caráter avaliativo para a sustituicao da P2 da disciplina de Estrutura de Dados.
Curso de Analise e Desenvolvimento de Sistemas ministrado pela Fatec Itapetining

## Setup
Clone o projeto do repositório https://github.com/wartocampos/pesq_clima_itape_1998_2018.git
Escolha sua IDE de preferencia e import o projeto.

### Gerenciador de Dependencias
Utilize o pip que fara a leitura do arquivo requirements.txt

#### PyCharm - Configuracao do projeto
Clique em File > New Project.
Em Location, selecione o local em que foi clonado o projeto.
Selecione a opcao New environment using Virtualenv.
Selecione o botao Create, e confirme a janela Create from Existing Sources.
Clique em File > Settings.
Selecione Project:pesq_clima_itape_1998_2018.
Clique com o botao direito em src.
Selecione Sources.
Clique em OK.

#### PyCharm - Configuracao de dependencias
Clique em File > Settings
Selecione Project:pesq_clima_itape_1998_2018
Selecione Python Interpreter
Abaixo de Python Interpreter, clique em + (Intall packages)
Faca uma busca por pandas e clique em Install Package
Faca uma busca por matplotlib e clique em Install Package
Feche as janelas e comece a codificar.
:)

#### PyCharm - Executando o programa
Clique com o boatao direito em main.py
Selecione a opcao Run 'main'.
:)
